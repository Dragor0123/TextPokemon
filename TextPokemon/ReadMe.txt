========================================================================
    콘솔 응용 프로그램 : TextPokemon 프로젝트 개요
========================================================================

Game Freak 사의 Pokemonster Red 버젼을 기반으로 만든
console에서 동작하는 Text RPG로 만든 포켓몬스터 게임입니다.

조작키는 키보드 방향키 ↑, ↓, ←, → 와 z,x,a,s 키입니다.

↑, ↓, ←, → : 상,하,좌,우 이동키

z : 확인 버튼 - 선택을 확인하거나, 사물, 대화 등을 체크하거나, 텍스트를 스크롤할 때 사용한다.
x : 취소 버튼 - 종료하거나, 선택을 취소하거나, 모드를 취소할 때 사용한다.
a : 시작 버튼 - MENU를 열 때 사용한다.
s : 선택 버튼 - 아이템을 바꾸거나 등록된 아이템을 사용할 때 사용한다.
